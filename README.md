# Vanilla JavaScript Web Application

This is a simple front-end web application built with no frameworks, using only vanilla JavaScript, HTML, and CSS. All files are placed in the root directory.

## Files
- `index.html`: Main HTML file
- `style.css`: CSS styles
- `script.js`: JavaScript logic

## Usage
Open `index.html` in your browser to view the app.

## Features
- Simple button click interaction using JavaScript
- Clean, modern styling

## No frameworks or build tools required
